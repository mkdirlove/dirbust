#### Installation and Usage:

Copy-paste this into your terminal:

```sh
sudo curl -fsSL https://raw.githubusercontent.com/mkdirlove/osfetch/main/osfetch -o /usr/bin/osfetch
```
```
sudo chmod +x /usr/bin/osfetch
```
```
osfetch
```
